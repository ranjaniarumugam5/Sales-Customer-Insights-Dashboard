# Sales & Customer Insights Dashboard

## Project Overview

This project presents an interactive Tableau dashboard designed to analyze sales performance, customer behavior, and business KPIs. The dashboard helps stakeholders monitor trends, identify opportunities, and make data-driven decisions.

---

## Tools Used

- Tableau
- Data Visualization
- Business Intelligence
- KPI Reporting

---

## Dashboard Features

- Sales Performance Analysis
- Customer Insights
- Profitability Trends
- Regional Performance Analysis
- Interactive Filters and Drill-downs
- KPI Tracking

---

## Key Metrics

- Total Sales
- Total Profit
- Customer Count
- Average Order Value
- Regional Sales Performance

---

## Business Value

The dashboard enables business stakeholders to:

- Monitor sales performance in real time
- Identify top-performing products and regions
- Understand customer purchasing behavior
- Support strategic decision-making through data-driven insights
